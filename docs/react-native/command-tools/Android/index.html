  • Commands

    • Start the emulator from the command line
      %LOCALAPPDATA%\Android\Sdk\emulator\emulator -list-avds
      %LOCALAPPDATA%\Android\Sdk\emulator\emulator -avd <avd> -writable-system -no-snapshot-load -no-boot-anim

  • Performance
